IsoplotR 5.0 features some minor changes on the surface (including more flexible ways to report output uncertainties), and some major changes under the hood. The new version has gone through beta testing, but it is possible that some issues remain. Please email p.vermeesch [at] ucl.ac.uk to report any broken functions, so that they can be fixed asap. The old version (4.4) can be accessed here

Vermeesch, P., 2018, IsoplotR: a free and open toolbox for geochronology. Geoscience Frontiers, v.9, p.1479-1493, doi:10.1016/j.gsf.2018.04.001.
